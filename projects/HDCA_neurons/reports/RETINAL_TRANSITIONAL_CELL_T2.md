# RETINAL_TRANSITIONAL_CELL_T2 — Retinal transitional cell, type 2

Cell Ontology: [retinal progenitor cell](http://purl.obolibrary.org/obo/CL_0002672) (CL:0002672, **broad match** — Same as T1 — transitional cell, not progenitor. NTR candidate.)

## Summary

T2 transitional cells in HDCA show a minimal RORB+/CRABP1+ signature, consistent with an intermediate neurogenic state with reduced progenitor-marker expression but not yet committed to a terminal phenotype.

## HDCA atlas evidence

**Cell count:** 1,018 cells. HDCA classifies this cluster as `broad_celltype = AUDIOVISUAL_NEURONAL`, `systems = AUDIOVISUAL_SYSTEM`.

### HDCA paper text


HDCA characterises the embryonic retina:

> "embryonic retina was characterised by expression of ATOH7, required for retinal ganglion cell (RGC) development71; PRDM13, which specifies amacrine cell fate in mouse72; GADD45A, which characterises early neurogenic cells in mouse retina73; and TRH, which encodes thyrotropin releasing hormone"
>
> — Webb et al. (2026), HDCA
