# RETINAL_TRANSITIONAL_CELL_T3 — Retinal transitional cell, type 3 (late-neurogenic, FABP7+)

Cell Ontology: [retinal progenitor cell](http://purl.obolibrary.org/obo/CL_0002672) (CL:0002672, **broad match** — Same as T1/T2 — FABP7+ late-born/Müller-biased transitional. NTR candidate.)

## Summary

T3 transitional cells co-express FABP7 (radial-glia-like) with RORB, consistent with the late-born / Müller-glia-biased transitional state described in human retinal scRNA-seq.

## HDCA atlas evidence

**Cell count:** 1,873 cells. HDCA classifies this cluster as `broad_celltype = AUDIOVISUAL_NEURONAL`, `systems = AUDIOVISUAL_SYSTEM`.

### HDCA paper text


HDCA characterises the embryonic retina:

> "embryonic retina was characterised by expression of ATOH7, required for retinal ganglion cell (RGC) development71; PRDM13, which specifies amacrine cell fate in mouse72; GADD45A, which characterises early neurogenic cells in mouse retina73; and TRH, which encodes thyrotropin releasing hormone"
>
> — Webb et al. (2026), HDCA
