# RETINAL_NEURAL_PROGENITOR — Retinal neural progenitor (committed-state neurogenic RPC)

Cell Ontology: [retinal progenitor cell](http://purl.obolibrary.org/obo/CL_0002672) (CL:0002672, **narrow match** — HDCA assigned CL:0002259 (neuroepithelial stem cell) but its CL definition is glial-precursor-focused and not retinal. Better fit: CL:0002672 narrow (HDCA RNP is a committed PAX6+/LHX2+ neurogenic-bia...)

## Summary

Retinal neural progenitors in HDCA represent a more committed/regionalised state of the retinal neuroepithelium, marked by PAX6, SOX2, SIX3 and the eye-field LHX2 transcription factor, with the secreted Wnt antagonist SFRP2 and the MYCN proliferation driver. This matches the canonical 'optic-cup PAX6+/LHX2+ RPC' state from which neurogenic divisions are launched.

## HDCA atlas evidence

**Cell count:** 767 cells. HDCA classifies this cluster as `broad_celltype = AUDIOVISUAL_NEUROEPITHELIUM`, `systems = AUDIOVISUAL_SYSTEM`.

HDCA characterises the embryonic retina via these progenitor and lineage markers:

> "embryonic retina was characterised by expression of ATOH7, required for retinal ganglion cell (RGC) development71; PRDM13, which specifies amacrine cell fate in mouse72; GADD45A, which characterises early neurogenic cells in mouse retina73; and TRH, which encodes thyrotropin releasing hormone"
>
> — Webb et al. (2026), HDCA
